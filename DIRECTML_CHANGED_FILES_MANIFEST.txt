VideOCR AMD DirectML v9 changed files
=====================================

Changed from v8:
- VideOCR.py
  - Adds a real DirectML GPU dropdown in Advanced Settings.
  - Adds a Refresh button for rescanning Windows display adapters.
  - Keeps the existing hidden numeric --directml_device_index CLI value synced.
  - Keeps DirectML grid width/height tuning from v8.
  - Autosaves DirectML grid settings.
- README.md
  - Documents the v9 DirectML GPU dropdown.
- README_AMD_DIRECTML.md
  - Documents the v9 DirectML GPU dropdown.

Notes:
- The dropdown is friendly GUI sugar over the same DirectML adapter index used by torch-directml.
- On the user's Ryzen+iGPU + RX 7900 XTX desktop, GPU 1 should normally be the RX 7900 XTX.
- Auto still falls back to the existing DirectML adapter candidate logic.
