v7 changes:
- DirectML adapter selection now prefers adapter index 1 on multi-GPU AMD systems.
- Supports forcing the DirectML adapter with VIDEOCR_DIRECTML_DEVICE_INDEX.
- Helper BAT files set VIDEOCR_DIRECTML_DEVICE_INDEX=1 for Ryzen+iGPU + RX 7900 XTX systems.
- DirectML diagnostics print the selected adapter index.
