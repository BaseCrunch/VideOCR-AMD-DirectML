v13 additions:
- New OCR engine option: ONNX Runtime DirectML (AMD GPU Experimental)
- New CLI engine value: --ocr_engine onnx_directml
- Optional ONNX dependencies in the DirectML extra: onnxruntime-directml and rapidocr-onnxruntime
- New backend file: CLI/videocr/onnx_directml_ocr.py
- ONNX backend tries RapidOCR + ONNX Runtime DmlExecutionProvider first
- If ONNX DirectML is unavailable or fails, it falls back to the working EasyOCR DirectML Hybrid backend instead of crashing
- Added GUI Last Run Benchmark panel
- Added [Perf] and [Bench] summary lines for Step 3, end-to-end runtime, and speed vs real-time
